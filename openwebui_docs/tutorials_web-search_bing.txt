Bing API​

Setup​

Navigate to the AzurePortal and create a new resource. After creation, you’ll be redirected to the resource overview page. From there, select "Click here to manage keys." On the key management page, locate Key1 or Key2 and copy your desired key. Open the Open WebUI Admin Panel, switch to the Settings tab, and then select Web Search. Enable the Web search option and set the Web Search Engine to bing. Fill SearchApi API Key with the API key that you copied in step 2 from AzurePortal dashboard. Click Save.
